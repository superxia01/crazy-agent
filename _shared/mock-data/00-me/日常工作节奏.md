# 日常工作节奏

## 每日时间表（工作日）

### 🌅 早上（8:00 - 10:00）
- **8:00-8:30** 浏览行业资讯（36氪、虎嗅、AI 相关公众号）
- **8:30-9:00** 回复客户微信和社群消息
- **9:00-9:15** 发布朋友圈（观点/金句类）
- **9:15-10:00** 深度工作 — 写内容/做方案/备课

### ☀️ 上午（10:00 - 12:00）
- **10:00-11:30** 客户沟通（电话/微信/视频）
  - A 级客户优先
  - 新线索首次破冰
- **11:30-12:00** 更新 CRM 备注，整理跟进计划

### 🍱 中午（12:00 - 14:00）
- **12:00-12:15** 发布朋友圈（案例/数据类）
- **12:00-13:00** 午饭 + 社群互动
- **13:00-14:00** 休息/碎片阅读

### 🌇 下午（14:00 - 18:00）
- **14:00-16:00** 核心交付时间
  - 内训备课 / 线上陪跑会议 / 方案撰写
  - 周二周四留给陪跑客户例会
- **16:00-17:00** 内容创作（选题/写作/排版）
- **17:00-18:00** 次日计划 + CRM 更新

### 🌙 晚上（20:00 - 21:30）
- **20:00-20:15** 发布朋友圈（生活/团队类）
- **20:15-21:00** 学习充电（AI 新工具测评、课程、读书）
- **21:00-21:30** 碎片灵感记录到 inbox

## 每周节奏

| 时间 | 重点 |
|------|------|
| 周一 | 规划本周内容选题 + 排期 |
| 周二 | 陪跑客户例会日（集中处理） |
| 周三 | 深度内容创作日 |
| 周四 | 陪跑客户例会日 + 客户回访 |
| 周五 | 社群运营 + CRM 清理 + 下周准备 |
| 周六 | 线下活动/沙龙（每月 1-2 次） |
| 周日 | 休息 + 灵感收集 |

## 每月节奏

- **每月 1 号**：复盘上月数据（营收、内容数据、客户转化）
- **每月 5 号**：发布当月内容排期
- **每月 15 号**：集中跟进行将过期的报价
- **每月 25 号**：社群主题分享/直播
- **每月最后一天**：下月计划和资源准备
