# IP定位

## 📌 填写指南
用一句话说清楚你是谁、帮谁解决什么问题。这是你所有内容的出发点。
定位清晰 = 受众精准 = 转化率高。定位模糊 = 什么人都想抓 = 什么人都抓不住。
如果暂时说不清楚，先填"我不做什么"——排除法也能帮你找到方向。

## ✏️ 你的内容

- 一句话定位：____________
- 人设标签（3个）：____、____、____
- 我不做的事：____________
- 我的独特经历/背书：____________

## 📖 参考：AI博主示例

- 一句话定位：教企业老板用AI降本增效的实战派
- 人设标签：反鸡汤、重数据、说人话
- 我不做的事：不卖焦虑、不割韭菜、不吹概念
- 独特经历：前跨境电商从业者，用AI把运营效率翻了5倍，现在做企业AI培训
