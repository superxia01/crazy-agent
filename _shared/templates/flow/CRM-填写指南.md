# CRM 客户关系管理系统

> 本目录用于管理完整的客户生命周期：从线索获取、客户管理、渠道分析到成交跟进和业绩统计。

## 📂 目录结构

| 目录 | 说明 |
|------|------|
| [10-线索管理](10-线索管理/) | 新线索录入与初步评估 |
| [20-客户管理](20-客户管理/) | 客户档案与关系维护 |
| [30-渠道管理](30-渠道管理/) | 获客渠道效果追踪 |
| [40-跟进记录](40-跟进记录/) | 客户跟进与沟通记录 |
| [50-成交记录](50-成交记录/) | 成交记录与回款管理 |
| [60-业绩统计](60-业绩统计/) | 月度业绩报表与分析 |

## 🔄 客户流转路径

```
新线索 (10-线索管理)
  ↓ 评估 & 资质确认
正式客户 (20-客户管理)
  ↓ 持续跟进 (40-跟进记录)
成交 (50-成交记录)
  ↓ 复盘 & 续费
业绩统计 (60-业绩统计)
```

## 📋 使用规范

1. **线索进入** → 使用 `10-线索管理/_template-线索卡.md` 创建线索卡
2. **线索转客户** → 使用 `20-客户管理/_template-客户档案.md` 建立完整档案
3. **每次跟进** → 使用 `40-跟进记录/_template-跟进记录.md` 记录沟通详情
4. **成交后** → 在 `50-成交记录/` 录入成交信息
5. **月末复盘** → 使用 `60-业绩统计/_template-月度业绩报表.md` 汇总数据

## 🏷️ 客户分层标准

| 等级 | 定义 | 跟进频率 |
|------|------|----------|
| A | 明确需求+预算+决策权，近期可成交 | 每周1-2次 |
| B | 有需求+预算，但决策周期较长 | 每两周1次 |
| C | 有潜在需求，需持续触达培养 | 每月1-2次 |
| D | 暂时不匹配，降低优先级 | 每季度1次 |

## 🌍 海外版说明

海外业务使用对应 `-海外版` 模板，字段包含 WhatsApp、LinkedIn、时区、沟通语言等海外专属信息。
