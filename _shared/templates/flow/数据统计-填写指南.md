# 📊 数据统计

存放公域流量各平台的数据汇总和趋势分析。

## 文件规范
- **命名**：`数据统计-YYYY-MM.md`（按月归档）
- **内容**：各平台关键指标（播放/阅读/粉丝增长/互动率）
- **格式**：Markdown 表格 + 简要分析

## 数据来源
- 各平台后台手动导出
- AI 定期汇总（需配置定时任务）

## 关键指标
| 平台 | 核心指标 |
|------|---------|
| 公众号 | 阅读、点赞、在看、转发、净增粉丝 |
| 小红书 | 浏览、点赞、收藏、评论 |
| 抖音/视频号 | 播放、完播率、点赞、评论、转发 |
| TikTok | Views、Engagement、Follower Growth |
| YouTube | Views、Watch Time、CTR、Subscribers |
| Facebook | Reach、Engagement、Clicks |
| Instagram | Reach、Saves、Shares、Engagement |
| Blog | Organic Traffic、Bounce Rate、Avg Time |
