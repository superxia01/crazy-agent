# 10-AI 智能层

本目录是整个 Vault 的 AI 能力中枢，存放所有与 AI 辅助相关的配置、模板、自动化规则和工具集成。

## 子目录说明

| 目录 | 用途 |
|------|------|
| `10-schemas/` | 数据结构与 Schema 定义 |
| `20-templates/` | AI 处理模板 |
| `20-调研分析/` | AI 调研与分析记录 |
| `30-dataview/` | Dataview 查询与视图 |
| `40-automations/` | 自动化脚本与工作流 |
| `40-效率工具/` | AI 效率工具配置 |
| `50-mcp/` | MCP 服务接入配置 |
| `60-prompts/` | Prompt 库 |
| `70-integrations/` | 外部集成配置 |
| `80-rules/` | AI 处理规则与约束 |
| `schedules/` | 定时任务与调度 |

## 管理者
AI 为主，人审核关键规则变更。

## 文件流转
- **输入**：从 `20-raw/` 获取原始数据，从 `60-events/` 获取事件触发
- **输出**：处理结果流向 `40-flow/` 各业务目录
