# 60-prompts Prompt 库

存放经过验证和优化的高质量 Prompt 模板，用于各类 AI 交互场景。

## 用途
- 内容创作 Prompt
- 数据分析 Prompt
- 代码生成 Prompt
- 翻译与润色 Prompt

## 文件命名
- 格式：`{场景}-{用途}.prompt.md`
- 示例：`内容-Blog写作.prompt.md`、`数据-周报生成.prompt.md`

## 管理者
**AI 初稿 → 人实测调优 → 入库**

## 文件流转
- **输入**：日常使用中积累的优质 Prompt
- **输出**：被 `40-automations/`、`20-templates/` 和手动操作引用
