## 2. 工作流程

### 2.1 新员工入职

1. 收集员工基本信息
2. 创建员工档案文件（`员工档案/姓名.md`）
3. 更新 `人事索引.md` 员工花名册
4. 添加考勤记录行到当月考勤表
5. 记录合同信息

### 2.2 月度考勤记录

1. 收集当月考勤数据（出勤/请假/加班/迟到）
2. 汇总到月度考勤文件
3. 计算异常情况
4. 反馈给相关管理者

### 2.3 季度绩效评估

1. 确定评估周期和参与人员
2. 填写各员工绩效评分
3. 汇总绩效排名
4. 生成绩效报告
5. 与员工沟通结果

### 2.4 员工离职

1. 更新员工档案状态为"已离职"
2. 记录离职日期和原因
3. 办理交接清单
4. 归档员工档案

## 4. 注意事项

1. **隐私保护**：员工档案含敏感信息，注意访问控制
2. **及时更新**：入职/离职/调岗应在3个工作日内更新档案
3. **考勤周期**：自然月为考勤周期，次月3号前完成上月汇总
4. **绩效公正**：绩效评估需多维数据支撑，避免主观偏见
5. **合同预警**：合同到期前30天提醒续签
6. **数据留存**：离职员工档案至少保留2年
