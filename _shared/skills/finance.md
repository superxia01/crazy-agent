## 2. 工作流程

### 2.1 记录一笔收支

当用户说"记一笔收入/支出"或类似指令时：

1. **确认信息**：向用户确认以下字段（缺省的主动询问）：
   - 日期（默认今天）
   - 类型：收入 / 支出
   - 金额（元）
   - 类别（见下方分类表）
   - 关联客户/供应商（如适用）
   - 备注

2. **写入月度收支文件**：追加到当月文件

3. **更新索引**：更新 `财务索引.md` 中的本月汇总行

### 2.2 生成月度利润报表

当用户说"生成本月利润报表"或类似指令时：

1. 读取当月收支文件中的所有记录
2. 按类别汇总收入和支出
3. 计算净利润 = 总收入 - 总支出
4. 与上月对比（如有上月数据）
5. 写入 `月度报表/2026/` 目录

### 2.3 管理发票

当用户说"记录发票"、"开发票"时：

1. 记录发票信息（发票号、金额、客户、日期、状态）
2. 更新 `发票登记簿.md`
3. 根据状态移动到"待开发票"或"已开发票"

## 5. 注意事项

1. **金额精度**：所有金额保留2位小数，单位统一为人民币元
2. **文件命名**：月度文件统一用 `YYYY-MM-收支.md` 格式
3. **及时更新**：每笔收支应在发生当天或次日记录
4. **月末结转**：每月1-3号完成上月报表生成
5. **备份**：财务数据属于核心数据，修改时注意保留历史版本
6. **权限**：财务文件涉及敏感信息，注意访问控制
