# 海外公域流量内容管理 Skill

## 1. 核心定位

本 Skill 是海外公域流量运营的中枢。以 **"内容驱动获客 → 社交证明转化"** 为主线，
通过 AI 协作实现选题→生产→发布→复盘的完整闭环。

**内容配比（黄金比例）：**

| 类型 | 占比 | 说明 |
|------|------|------|
| 🛠️ 产品实操 & 教程 | **50%** | 开箱测评、使用教程、对比评测、FAQ 解答、安装指南 |
| 🧠 行业认知 & 趋势 | **20%** | 行业报告解读、市场趋势分析、供应链洞察、政策变化、选品思路 |
| 💬 客户见证 & 晒单 | **15%** | 真实买家评价、订单截图、Before/After、UGC 内容、Review 合集 |
| 📸 日常真实 & 幕后 | **15%** | 工厂/仓库/办公室日常、团队故事、打包发货、打包过程、创业感悟 |

## 2. 覆盖平台 & 内容形式

### 2.1 Facebook
- **长图文帖（Long Post）**：500-1500 词，深度行业分析、产品故事、品牌理念。适合沉淀专业形象。
- **轮播帖（Carousel）**：5-10 张卡片，产品系列展示、功能对比、步骤教程。高互动形式。
- **视频（Video）**：1-5 分钟，产品演示、客户采访、Behind the Scenes。原生视频优先于链接分享。
- **Stories**：24h 限时内容，促销预告、直播倒计时、日常花絮、投票互动。保持每日 2-5 条更新。
- **群组（Groups）**：垂直社群运营（如跨境电商交流群、产品测评群），以价值输出建立 KOL 权威。

> **Facebook 最佳发布时间**（目标市场美区）：周二-周四 9AM-1PM EST；周末互动率高但 B2B 转化低。

### 2.2 Instagram
- **图文帖（Feed Post）**：1:1 或 4:5 比例，高质量产品图 + 150-300 词 caption。每篇 3-5 个相关 hashtag + 2 个品牌专属 hashtag。
- **轮播帖（Carousel）**：最多 10 张，教程步骤、产品细节放大、客户故事。首图必须足够吸睛（Hook 首图）。
- **Reels**：15-90 秒竖屏视频，产品演示、趋势挑战（Trending Audio）、快速教程、开箱反应。**目前 Instagram 流量增长最快的形式。**
- **Stories**：每日 3-8 条，产品近景、限时优惠、Question Box 互动、Link Sticker 导流独立站。
- **Guides**：聚合类内容，将已有帖子/Reels 整理成主题合集（如"新手选品指南"）。

> **Instagram 关键算法信号**：Saves（收藏）> Shares（分享）> Comments（评论）> Likes（点赞）。内容策略应优先引导收藏和转发。

### 2.3 TikTok
- **短视频（Short Video）**：15-60 秒，竖屏 9:16。核心公式：**3秒 Hook → 价值交付 → CTA**。
- **教育类内容（Educational）**：行业知识科普、选品技巧、平台运营心得、"原来这个东西这么好用"类种草。
- **产品种草（Product Showcase）**：真实使用场景展示，不做硬广。追求"不经意的推荐感"。
- **开箱测评（Unboxing）**：第一人称视角开箱，配合真实反应和即兴点评。
- **幕后花絮（Behind the Scenes）**：仓库打包、工厂验货、展会现场、发货量展示。

> **TikTok 流量密码**：前 3 秒决定留存率；BGM 选择当前 Trending Audio；文案控制在 150 字以内；发布时间本地晚间 7-10PM。

### 2.4 YouTube
- **长视频（Long-form）**：5-15 分钟，横屏 16:9。结构：Hook（15s）→ 价值预告（30s）→ 正文 → 总结 → CTA。适合产品深度测评、行业分析、教程系列。
- **Shorts**：15-60 秒竖屏，作为长视频的引流入口。可在 Short 中预告长视频内容并引导跳转。
- **内容系列化（Playlist）**：将相关视频组织成系列（如"Shopify 建站从零到一"、"亚马逊 PPC 精通课"），提高频道粘性。
- **SEO 优化**：标题含核心关键词、描述 200+ 词含长尾关键词、标签 5-15 个、自定义缩略图（CTR 提升核心）。

> **YouTube 核心指标**：CTR（缩略图+标题决定）→ 平均观看时长（内容质量决定）→ 订阅转化。

### 2.5 Google Blog (SEO)
- **SEO 长文（Blog Post）**：1500-3000 词，深度覆盖一个核心主题。结构：H1 标题 → 目录 → H2/H3 分层 → 内链 → CTA。
- **关键词策略**：1 个主关键词 + 3-5 个次关键词。工具推荐：Ahrefs / SEMrush / Google Keyword Planner / Ubersuggest。
- **内容类型**：How-to 指南、产品对比评测（vs.）、Best of 合集（Best [product] for [use case]）、行业报告。
- **更新策略**：已发布的 SEO 文章每 3-6 个月更新一次，添加新数据、修正过时信息，保持排名。

> **Blog SEO 核心**：Search Intent 匹配 > 内容深度 > 内链结构 > 页面速度。不为写而写，每篇文章解决一个明确的搜索意图。

## 3. 工作流程

### 阶段一：理解用户

当用户触发本 Skill 时，首先确认以下信息（可从对话上下文或 vault 配置中获取）：

```
□ 用户身份：跨境电商卖家 / 外贸 SOHO / 品牌方 / 服务商
□ 目标市场：北美 / 欧洲 / 东南亚 / 拉美 / 中东
□ 产品类目：3C / 服饰 / 家居 / 美妆 / 工具 / 其他
□ 主营平台（销售端）：Amazon / Shopify / 独立站 / 速卖通 / Temu / 其他
□ 内容运营阶段：冷启动 / 增长期 / 成熟期
□ 团队规模：个人 / 2-3人小团队 / 有专职内容团队
□ 当前痛点：缺选题 / 生产效率低 / 数据差 / 没方向
```

> 如信息不足，**主动询问 1-2 个关键问题**，不要猜测。优先级：目标市场 > 产品类目 > 运营阶段。

### 阶段二：选题生成

#### 2.1 信息扫描

扫描 vault 中的以下目录，提取选题素材：

| 扫描目录 | 扫描目的 | 扫描方式 |
|----------|----------|----------|
| `20-raw/` | 用户最近的输入记录、对话、灵感碎片 | 按时间倒序读取最近 20 个文件 |
| `30-wiki/` | 产品知识库、行业知识、FAQ | 读取所有 markdown，提取可外化知识点 |
| `40-flow/` | 业务流程中的可分享节点 | 扫描流程文件中的关键步骤 |
| `40-flow/` | 当前项目进展（新品上架、活动策划） | 读取项目状态，提取可宣传节点 |

#### 2.2 选题生成方法

基于扫描结果 + 内容配比，生成选题列表。每个选题包含：

```yaml
topic:
  title: "具体标题"
  category: "产品实操 | 行业认知 | 客户见证 | 日常真实"
  platform: "目标发布平台"
  format: "内容形式"
  hook: "一句话吸睛点"
  target_keyword: "SEO 关键词（Blog 专用）"
  cta: "引导动作"
  priority: "高 | 中 | 低"
```

**选题来源矩阵：**

| 素材来源 | 选题方向 | 适合平台 |
|----------|----------|----------|
| 客户高频提问 (wiki/FAQ) | 教程类内容 | YouTube, Blog, TikTok |
| 新品上架计划 | 开箱/测评 | Instagram, TikTok, YouTube |
| 热销品数据 | 种草/教程 | Facebook, Instagram, TikTok |
| 行业新闻/报告 | 趋势解读 | Facebook, Blog, YouTube |
| 订单截图/好评 | 客户见证 | Instagram Stories, Facebook |
| 仓库/工厂实拍 | 幕后内容 | TikTok, Instagram Stories |
| 竞品分析 | 对比评测 | YouTube, Blog |
| 节日/促销节点 | 活动预热 | 全平台 |

#### 2.3 选题输出格式

为用户输出结构化选题列表，按优先级排序：

```markdown
## 📋 本周选题推荐

### 高优先级
1. **[TikTok Reel]** "这个 $19 的小东西，美国客户已经下了 3000 单"
   - 类型：产品实操
   - Hook：价格 + 销量数字冲击
   - 拍摄建议：手持展示 + 订单截图闪现

2. **[Blog SEO]** "Best Dropshipping Products 2026: Complete Guide"
   - 类型：行业认知
   - 目标关键词：dropshipping products 2026
   - 预计字数：2500 词

### 中优先级
3. **[Instagram Carousel]** "5 Reasons Our Water Bottle Beats Yeti"
   - 类型：产品实操
   - 形式：5 张轮播对比图

4. **[YouTube]** "Full Amazon FBA Walkthrough: $0 to $10K/Month"
   - 类型：行业认知
   - 时长：12 分钟
```

### 阶段三：内容生产

根据用户确认的选题，协助完成内容创作：

**视频类（TikTok/YouTube/Instagram Reels）：**
- 输出：脚本（Hook → 正文 → CTA）、分镜建议、BGM 推荐方向、拍摄清单
- 格式：带时间轴的脚本表格

**图文类（Facebook/Instagram Feed）：**
- 输出：完整 Caption 文案、Hashtag 组合、图片拍摄指导（构图/角度/场景）
- Facebook 长文需含段落分隔 + 小标题

**SEO 博客（Blog）：**
- 输出：完整文章（1500-3000 词）、SEO 元数据（Title Tag、Meta Description、关键词密度）、内链建议、图片插入点标注
- 结构：H1 → 引言 → 目录 → H2 分段 → 总结 → CTA → FAQ Schema

### 阶段四：发布状态管理

每个内容 piece 都应跟踪其生命周期状态。

## 5. 状态流转

每个内容 piece 必须标记状态，状态单向流转（可回退）：

```
📋Idea → 📝Draft → 🎥Filming → ✂️Editing → 📤Ready → ✅Published → 🚫Closed
  │         │          │          │          │          │
  └─────────┴──────────┴──────────┴──────────┴──────────┘
                    （任意状态可回退修改）
```

**各状态说明：**

| 状态 | Emoji | 含义 | 负责人动作 |
|------|-------|------|-----------|
| Idea | 📋 | 选题通过，进入排期 | 确定选题、分配平台、排入日历 |
| Draft | 📝 | 脚本/文案初稿完成 | 撰写脚本、文案、SEO 大纲 |
| Filming | 🎥 | 拍摄/录制中 | 视频拍摄、图片拍摄、录音 |
| Editing | ✂️ | 后期制作中 | 剪辑、配字幕、调色、SEO 优化 |
| Ready | 📤 | 内容完成待发布 | 最终审核、定时发布设置 |
| Published | ✅ | 已发布上线 | 记录发布时间、初始数据 |
| Closed | 🚫 | 不再推广 | 数据已归档、经验已提炼 |

> **AI 协作重点**：在 Idea→Draft 阶段提供最大协助（选题扩展、脚本撰写、SEO 优化）。
> Draft→Filming→Editing 阶段提供指导建议（拍摄指导、剪辑节奏建议）。
> Published→Closed 阶段提供数据分析和优化建议。

## 7. 表现评级体系

根据各平台核心指标综合评定：

### 评级标准

| 评级 | 含义 | 标准（以 TikTok 为例） |
|------|------|----------------------|
| 🚀 **Viral** | 爆款，超出预期 5x+ | 完播率 >25%, 互动率 >8%, 播放量 >100K |
| **S** | 优秀，显著高于均值 | 完播率 >18%, 互动率 >5%, 播放量 >50K |
| **A** | 良好，高于均值 | 完播率 >12%, 互动率 >3%, 播放量 >20K |
| **B** | 正常，达到均值 | 完播率 >8%, 互动率 >1.5%, 播放量 >5K |
| **C** | 低于预期，需优化 | 完播率 <8%, 互动率 <1.5%, 播放量 <5K |

**各平台评级参考基线：**

| 平台 | 互动率均值 | 良好线 (A) | 优秀线 (S) | 爆款线 (Viral) |
|------|-----------|-----------|-----------|---------------|
| Facebook | 0.5-1% | >2% | >5% | >10% |
| Instagram Feed | 1-3% | >4% | >7% | >15% |
| Instagram Reels | 3-6% | >8% | >12% | >20% |
| TikTok | 3-8% | >8% | >15% | >25% |
| YouTube (长视频) | N/A (看时长) | 时长>50% | 时长>65% | 时长>80% |
| YouTube Shorts | 3-6% | >8% | >12% | >20% |
| Blog | 跳出率<70% | 停留>3min | 停留>5min | Top 3 排名 |

> **评级用途**：
> - S/Viral → 提炼成功模式，复制到其他平台/内容
> - A → 保持节奏，微调优化
> - B → 分析瓶颈点，针对性改进
> - C → 暂停同类内容，重新审视选题和形式

## 8. 内容日历管理

### 排期原则

| 原则 | 说明 |
|------|------|
| **频率稳定** | 每个平台维持稳定更新频率，不要三天打鱼 |
| **形式交替** | 同一平台避免连续发布同一形式（如连续两天都是 Reels） |
| **选题平衡** | 每周检查内容配比是否接近 50/20/15/15 |
| **节点对齐** | 营销活动（Prime Day、黑五、节日）提前 2-3 周排预热内容 |
| **跨平台复用** | 一个选题可适配多平台，但必须调整形式（非简单搬运） |

### 各平台推荐发布频率

| 平台 | 最低频率 | 推荐频率 | 高增长频率 |
|------|----------|----------|-----------|
| Facebook | 3 次/周 | 5-7 次/周 | 10+ 次/周 |
| Instagram Feed | 3 次/周 | 5 次/周 | 7 次/周 |
| Instagram Reels | 3 次/周 | 5 次/周 | 7+ 次/周 |
| Instagram Stories | 5 次/天 | 8 次/天 | 10+ 次/天 |
| TikTok | 3 次/周 | 5-7 次/周 | 14+ 次/周 |
| YouTube 长视频 | 1 次/周 | 2 次/周 | 3+ 次/周 |
| YouTube Shorts | 2 次/周 | 3-4 次/周 | 7+ 次/周 |
| Blog | 1 篇/周 | 2 篇/周 | 3+ 篇/周 |

## 9. 跨平台内容复用策略

**一鱼多吃原则**：一个核心选题 → 多平台适配，但必须重新制作而非直接搬运。

```
核心选题："我们的防水背包如何在暴雨中保护你的电脑"

├── TikTok (30s)    → 短视频：快节奏暴雨测试画面 + 音效 + 文字叠加
├── Instagram Reel  → 60s版：增加美感镜头 + 品牌配色字幕
├── Instagram Feed  → 轮播图：5 张关键帧 + 详细功能说明 caption
├── Facebook Video  → 2min版：完整测试过程 + 口播解说 + CTA 到独立站
├── YouTube         → 8min长版：专业防水测试 + 竞品对比 + 优惠信息
└── Blog            → 2000词文章："How Waterproof Backpacks Work: Our Lab Test Results"
```

**复用注意事项：**
- ❌ 不要把 TikTok 水印视频发到 Instagram Reels（会被算法降权）
- ❌ 不要用同一文案复制粘贴到所有平台（每个平台调性不同）
- ✅ 每个平台调整 Hook、时长、字幕风格、CTA
- ✅ TikTok/Shorts 先发测试数据，数据好的选题再扩展到长视频/Blog

## 10. 常见坑 & 避坑指南

### 🚨 致命错误（直接导致 0 流量）

| 坑 | 说明 | 正确做法 |
|----|------|----------|
| **搬运搬运搬运** | 同一内容不加修改发 5 个平台 | 每个平台重新适配形式、Hook、CTA |
| **硬广推销** | 每条内容都在卖东西，毫无价值 | 80% 提供价值，20% 软性引导 |
| **忽视 SEO 基础** | YouTube/Blog 完全不考虑关键词 | 每篇内容至少确定 1 个目标关键词 |
| **不研究竞品** | 闭门造车，不看同赛道爆款 | 每周分析 5-10 个竞品爆款内容 |
| **三天打鱼** | 兴致来了连发一周，然后消失一个月 | 设定可执行的最低频率并坚持 |

### ⚠️ 高频错误（显著影响效果）

| 坑 | 说明 | 正确做法 |
|----|------|----------|
| **Hook 太弱** | 前 3 秒无法抓住注意力 | 用数据/问题/反差/悬念 开头 |
| **没有 CTA** | 看完就走了，没有下一步引导 | 每条内容都有明确的 1 个 CTA |
| **Hashtag 滥用** | Instagram 塞 30 个无关标签 | 5-10 个精准标签 + 2 个品牌标签 |
| **发布时间随意** | 目标市场凌晨 3 点发帖 | 根据目标市场时区排定发布时间 |
| **不做数据分析** | 发完就忘，不知道什么有效 | 发布后 24h/72h/7d 记录并复盘 |
| **封面/缩略图敷衍** | YouTube 直接用视频截图做缩略图 | 专门设计高 CTR 缩略图（大字+人脸+对比色） |
| **语言不地道** | 明显的翻译腔/中式英语 | 使用母语者或 AI 润色，符合目标市场表达习惯 |

### 💡 优化技巧（从 B 提到 A/S）

1. **TikTok 视频开头加入"意外画面"**（如摔落测试、极端条件测试），提高完播率
2. **Instagram 轮播首图使用大字标题**（如"STOP SCROLLING"），提高进入率
3. **YouTube 标题使用数字+情绪词**（如"I Made $50K Selling on Amazon – Here's How"），提高 CTR
4. **Blog 文章添加 FAQ Schema**，增加搜索结果展示面积
5. **Facebook 视频使用原生上传**而非 YouTube 链接，算法给予更高权重
6. **所有平台统一品牌视觉**（Logo、配色、字体），建立辨识度
7. **利用 UGC（用户生成内容）**：鼓励客户发使用视频/照片，授权后在官方账号发布
8. **TikTok 使用 Green Screen / Duet / Stitch** 功能蹭热度

## 11. AI 协作模式

当用户触发本 Skill 时，按以下模式工作：

### 模式一：选题生成
> 用户说："帮我生成下周的选题"

**AI 动作**：
1. 扫描 `20-raw/` 和 `30-wiki/` 获取素材
2. 按 50/20/15/15 比例生成本周选题
3. 每个选题标注平台、形式、优先级
4. 输出结构化选题列表

### 模式二：内容创作
> 用户说："帮我写一个 TikTok 脚本，主题是 xxx"

**AI 动作**：
1. 确认平台、时长、目标受众
2. 输出带时间轴的脚本（Hook → Value → CTA）
3. 提供拍摄建议（场景、道具、BGM 风格）
4. 建议 3 个 Hashtag 组合

### 模式三：SEO 文章
> 用户说："帮我写一篇关于 xxx 的 SEO 文章"

**AI 动作**：
1. 建议目标关键词和次关键词
2. 输出完整文章（1500-3000 词）
3. 包含 SEO 元数据（Title Tag、Meta Description）
4. 标注内链插入点和图片建议

### 模式四：数据复盘
> 用户说："帮我分析这周的数据"

**AI 动作**：
1. 读取各平台数据记录
2. 按评级标准（Viral/S/A/B/C）分类
3. 提炼 Top 3 成功经验和 Top 3 改进方向
4. 基于数据调整下周选题建议

### 模式五：内容日历
> 用户说："帮我排这周的内容日历"

**AI 动作**：
1. 读取待发布内容（Status = Ready）
2. 按平台频率和配比排期
3. 标注发布时间和目标市场时区
4. 输出周计划表格

## 12. 快速参考

### 内容配比速查
```
🛠️ 产品实操  50%  ← 核心驱动，每周至少一半
🧠 行业认知  20%  ← 建立专业形象
💬 客户见证  15%  ← 社交证明，信任建设
📸 日常真实  15%  ← 拉近距离，人设打造
```

### 状态速查
```
📋 Idea     → 选题确认，进入排期
📝 Draft    → 脚本/文案完成
🎥 Filming  → 拍摄录制中
✂️ Editing  → 后期制作中
📤 Ready    → 待发布
✅ Published → 已发布
🚫 Closed   → 已归档
```

### 评级速查
```
🚀 Viral  → 爆款，5x 以上，提炼复制
S         → 优秀，2-5x，保持节奏
A         → 良好，1-2x，微调优化
B         → 正常，均值水平，稳步提升
C         → 低于预期，停复盘改
```

### 发布频率速查
```
TikTok:        每天 1 条起步
Instagram:     Reels 5条/周 + Stories 每天
Facebook:      5条/周
YouTube:       长视频 2条/周 + Shorts 3条/周
Blog/SEO:      2篇/周
```
