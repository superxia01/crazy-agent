# Crazy Agent 🚀

> **Hermes + Obsidian = 你的公司级操作系统。**

一套为「一个人就是一家公司」设计的 Obsidian 知识库模板 + Hermes Agent 配置包。你负责想，AI 负责干。

- 🇨🇳 **国内版**：公众号、小红书、抖音、微信私域
- 🌍 **海外版**：Facebook、Instagram、TikTok、YouTube、WhatsApp

---

## ✨ 这是什么？

**不是笔记软件，是一整套公司操作系统。**

想象一下你开了一家公司，各部门都需要管理：

| 部门 | 系统帮你管什么 |
|------|--------------|
| 市场部 | 选题、写内容、多平台发布、数据追踪 |
| 销售部 | 客户线索、跟进记录、成交归档、渠道分析 |
| 私域部 | 朋友圈、社群、数据周报 |
| 人事部 | 员工档案、考勤、绩效 |
| 财务部 | 收支记录、月度利润、发票 |
| 项目部 | 项目进度、任务分配、里程碑 |
| 仓库 | SKU 管理、订单、库存预警 |

**你不需要学任何软件，跟 AI 说话就行。**

> "帮我出 5 个选题"
> "新来一个客户，叫张伟"
> "记一笔收入，3 万块"
> "这篇已发布到小红书了"

AI 自动放到正确的地方，用正确的格式。

## 🏗️ 系统架构

```
你的公司/
├── 00-me/        🧠 你的脑子 — 定位、产品、目标
├── 10-AI/        🤖 你的 AI 员工 — prompt、规则、自动化
├── 20-raw/       📥 收件箱 — 看到好东西先扔进来
├── 30-wiki/      📚 知识库 — 消化后的结论
├── 40-flow/      🔁 各部门 — 日常操作台
├── 50-assets/    💰 资产 — 品牌、账号、合同、报价方案
├── 60-events/    📋 日志 — 公司大事 + AI 犯错记录
└── 99-archive/   🗄️ 档案室 — 已结束的东西
```

### 三条铁律

| 层 | 规则 |
|----|------|
| `20-raw` | **只进不改** — 素材扔进去就不动了 |
| `30-wiki` | **只放结论** — 只写自己消化后的话 |
| `40-flow` | **只做调用** — 日常工作只调用知识，不写知识 |

### 信息流水线

```
看到好内容 → 扔进 20-raw → 消化提炼到 30-wiki → 日常产出在 40-flow → 好东西沉淀到 50-assets
```

## 🚀 快速开始

### 前置要求

- [Hermes Agent](https://github.com/nousresearch/hermes-agent) 已安装并配置
- 终端或消息平台已连接 Hermes

### 一键搭建

告诉 Hermes：

**国内版：**
> 请按照 crazy-agent 国内版搭建我的 Vault

**海外版：**
> 请按照 crazy-agent 海外版搭建我的 Vault

Hermes 会自动创建全部目录、模板、Skills 和定时任务。搭完就能用。

### 不用 Hermes？

纯 Obsidian 也能用。参照 [GUIDE.md](GUIDE.md) 手动搭建。

## 📖 上手三步走

### 第 1 步：填好你的"脑子"

打开 `00-me/`，里面有 5 个子目录、11 个配置文件，每个都是三段式模板（填写指南 + 你的内容 + 行业示例）：

| 填什么 | 为什么重要 |
|--------|-----------|
| IP定位、品牌风格、个人故事 | AI 写所有内容都参考这个 |
| 产品清单、价格体系、交付标准 | AI 跟客户谈单时参考 |
| 客户画像、分层标准 | AI 帮你精准获客和分优先级 |
| 平台账号信息 | AI 帮你管多平台发布 |
| 商业目标、日常节奏 | AI 定时提醒你按节奏走 |

**这是根基。00-me 填得越详细，AI 干得越好。**

### 第 2 步：扔素材

把你收藏的文章、截图、竞品资料、行业报告，全部扔进 `20-raw/inbox/`。

不需要整理，一股脑扔就行。后续跟 AI 说"帮我整理素材"，自动消化。

### 第 3 步：开始干活

> "帮我出选题" → AI 参考 00-me 和 wiki 生成选题卡
> "写一篇公众号文章" → AI 按模板写草稿
> "新客户，张伟，华创科技" → AI 建客户卡
> "发朋友圈" → AI 参考品牌风格写文案
> "记一笔收入" → AI 自动记账

## 🤖 你说什么，AI 做什么

| 你说 | AI 做 |

| "帮我出选题" | 参考 00-me + wiki，生成选题卡 |

## ❓ 常见问题

**文件太多找不到？**
跟 AI 说"帮我找一下关于xxx的内容"，或在 Obsidian 里 Ctrl+O 搜索。

**素材越堆越多？**
跟 AI 说"帮我整理素材"，自动消化到 wiki。

**客户成交了要移文件吗？**
不需要。说"成交了"，AI 自动改 ✅ 标记，文件不搬家。

**国内版能切海外版吗？**
可以，跟 AI 说"按海外版重新搭建"，会保留你的 00-me 数据。

**Mock 数据是什么？**
演示用的假数据，帮你快速理解系统。不需要就说"删掉 mock 数据"。
